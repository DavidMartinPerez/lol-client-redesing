# Lol Modern Design

Rediseño de la aplicación de escritorio de league of legends en React+Eletron

## Preview
![Imagen previuw de la app](./docs/preview.png)

## Inicializar Proyecto
```cmd
npm install
npm run electron
```

## Stack
* React + Styled Components
* Electron
